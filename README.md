# Cola Personal Site Generator

根据用户的记忆、对话风格和性格，一键生成高质量个人品牌网站并部署到 GitHub Pages。

## 特点

- **25 种风格**：从硬核科技到日式侘寂，覆盖 5 大类审美方向
- **性格推断**：根据对话历史自动匹配最适合的风格
- **记忆驱动**：从 Cola 记忆中提取信息，不用手动填表
- **一键部署**：生成后自动推送到 GitHub Pages
- **拒绝 AI 通病**：禁用 Inter 字体、禁止紫色渐变白底、每个站都独一无二

## 安装

在 Cola 中安装此 Skill：

```
安装 skill：heran11011/cola-personal-site
```

## 使用

对 Cola 说：

- 「帮我做个个人网站」
- 「我想要一个个人品牌站」
- 「给我搭个主页」

Cola 会从记忆中提取你的信息，推断适合你的风格，确认方向后全自动生成 + 部署。
